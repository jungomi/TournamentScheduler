#README

## Usage
```
  java -jar TurnierPlanung.jar [<game_file> <spacing_file> <time_limit> <neighbor_range> <num_shuffle>]
  
  <game_file>
    path to the file with game informations
  <spacing_file>
    path to the file with spacing informations
  <time_limit>
    time limit in seconds
  <neighbor_range>
    range for the neighborhood generation
  <num_shuffle>
    number of shuffles done to find a new starting sequence
```
